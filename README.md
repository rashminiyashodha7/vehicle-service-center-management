# vehicle-service-center-management
This project is a service management system designed to streamline booking processes and operational workflows in vehicle service centers. It replaces manual processes with an automated solution that improves efficiency, visibility, and data handling.

The system supports service booking, operational tracking, and workflow management. It was developed with a strong focus on business process analysis and re-engineering, ensuring improved service delivery and operational consistency.

Tech Stack: MERN Stack, Flutter, Selenium, Git, GitHub
